node.js을 사용한 팀 프로젝트관련 프로그램이다.

내용 
팀을 추가하여 팀 페이지를 생성한다. 
팀 추가, 팀 삭제, 팀명 변경이 가능하고 팀원 추가, 삭제 기능이 존재한다.




테스트 방법을 아래 기재합니다.
1. Node JS설치합니다.
2. Visual Studio Code을 설치합니다.
3. mysql을 설치합니다.
4. mysql에 db.txt의 내용을 실행하여 데이터베이스, 테이블을 제작합니다. (초기 테이블 내용역시 입력합니다.)
5. visual Studio Code에서 파일-폴더열기를 통해 zip파일을 푼 폴더를 선택합니다.(db_main.js가 있는 폴더까지 들어가야합니다.)
6. 탐색기를 보면 lib폴더안에 db_db.js란 파일을 열어 mysql설치시 만든 user, password를 입력합니다.
7. 터미널 창을 열어 npm install http, npm install mysql을 설치해줍니다.
8. 터미널 창에서 node db_node.js을 입력하여 프로그램을 실행합니다.
9. 인터넷창을 켜서 localhost:3000으로 접속합니다.
10. 정상적으로 켜졌다면, 웹페이지 동작을 테스트할 수 있습니다.
